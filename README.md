# 🚀 Implantação Contínua (Continuous Deployment)

## 📌 Sobre o Projeto

Este projeto demonstra a implementação de um pipeline de **Implantação Contínua (Continuous Deployment)** utilizando GitHub Actions para um site estático.

## 🔄 Diferença para Entrega Contínua

| Característica | Entrega Contínua | Implantação Contínua |
|----------------|------------------|----------------------|
| Aprovação manual | ✅ Requer | ❌ Não requer |
| Deploy automático | ❌ Não | ✅ Sim |
| Intervenção humana | Sim, para produção | Zero intervenção |
